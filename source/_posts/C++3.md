## explict
防止类的构造函数的隐式转换
```cpp
class A
{
private:
    int x;
public:
    A(int x_): x{x_} {}
};

// ...

A a = 3; // YES
A a2{3}; // YES
```
这样是对的

```cpp
class A
{
private:
    int x;
public:
    explict A(int x_): x{x_} {}
};

// ...

A a = 3; // NO
A a{3}; //YES
```
